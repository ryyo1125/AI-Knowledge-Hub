# ネクストアップデートアイデア

**作成日**: 2026-02-04
**ステータス**: アイデア段階（未検討）

---

## 概要

ユーザー視点で「あったら嬉しい」機能のアイデア集。
MVPリリース後の機能拡張候補として検討する。

---

## アイデア一覧

### 1. 🎯 パーソナライズ推薦

**嬉しいポイント**: 自分に合った記事が自動で上に来る

| 項目 | 内容 |
|------|------|
| 機能概要 | ユーザーの好みに合わせた記事推薦 |
| 実装イメージ | スキルレベル設定、興味トピック登録、閲覧履歴学習 |
| 優先度 | 未定 |
| 難易度 | 中〜高 |

**詳細**:
- 自分のスキルレベル（初心者/中級/上級）を設定
- 興味のあるトピック（プロンプト設計、MCP、エージェント等）を登録
- 過去の閲覧・評価履歴から好みを学習

---

### 2. 📌 「後で読む」リスト

**嬉しいポイント**: 気になった記事を忘れない

| 項目 | 内容 |
|------|------|
| 機能概要 | 記事の一時保存・管理機能 |
| 実装イメージ | ブックマークボタン、未読/既読管理、リマインダー |
| 優先度 | 未定 |
| 難易度 | 低 |

**詳細**:
- ワンクリックで保存
- 未読/既読の管理
- 「1週間放置したら通知」オプション

---

### 3. 🔔 重要記事アラート

**嬉しいポイント**: 見逃したくない情報だけ通知される

| 項目 | 内容 |
|------|------|
| 機能概要 | 条件に合う記事の自動通知 |
| 実装イメージ | スコア閾値設定、キーワードフィルター、通知チャネル選択 |
| 優先度 | 未定 |
| 難易度 | 中 |

**詳細**:
- 「重要度4以上の記事が来たら通知」
- 「〇〇というキーワードを含む記事」
- 通知手段: メール / Slack / ブラウザ通知

---

### 4. 📊 学習ダッシュボード

**嬉しいポイント**: 自分の成長が可視化される

| 項目 | 内容 |
|------|------|
| 機能概要 | 学習進捗の可視化・統計表示 |
| 実装イメージ | 読了数グラフ、カテゴリ別進捗、成長サマリー |
| 優先度 | 未定 |
| 難易度 | 中 |

**詳細**:
- 今週読んだ記事数
- カテゴリ別の読了状況
- 「今月のあなた: プロンプト設計に強くなりました」

---

### 5. ✅ 「試してみた」チェック機能

**嬉しいポイント**: 知識→実践の橋渡し

| 項目 | 内容 |
|------|------|
| 機能概要 | 記事の実践状況トラッキング |
| 実装イメージ | ステータス管理（読んだ/試した/習得）、メモ機能 |
| 優先度 | 未定 |
| 難易度 | 低〜中 |

**詳細**:
- 記事ごとに「読んだ」「試した」「習得した」ステータス
- 試した結果のメモを残せる
- 「試したいリスト」の管理

---

### 6. 🔗 関連記事の自動リンク

**嬉しいポイント**: 芋づる式に学べる

| 項目 | 内容 |
|------|------|
| 機能概要 | 記事間の関連性を自動検出・表示 |
| 実装イメージ | タグ/トピック類似度、閲覧パターン分析 |
| 優先度 | 未定 |
| 難易度 | 中〜高 |

**詳細**:
- 「この記事を読んだ人はこちらも読んでいます」
- 同じトピックの過去記事を時系列で表示
- 前提知識となる記事への導線

---

### 7. 💬 AIに質問できる機能

**嬉しいポイント**: 記事を読んで生じた疑問をすぐ解決

| 項目 | 内容 |
|------|------|
| 機能概要 | 記事内容をコンテキストにしたAIチャット |
| 実装イメージ | 記事詳細画面にチャットUI、Claude API連携 |
| 優先度 | 未定 |
| 難易度 | 中 |

**詳細**:
- 「この記事の〇〇の部分、もう少し詳しく教えて」
- 「自分のプロジェクトに適用するにはどうすれば？」
- 記事内容をコンテキストにしたAIチャット

---

### 8. 📈 トレンドレーダー

**嬉しいポイント**: 今何が熱いかが一目でわかる

| 項目 | 内容 |
|------|------|
| 機能概要 | トピックのトレンド分析・可視化 |
| 実装イメージ | ワードクラウド、急上昇検出、時系列比較 |
| 優先度 | 未定 |
| 難易度 | 中〜高 |

**詳細**:
- 今週話題のキーワードをワードクラウド表示
- 「急上昇トピック」の通知
- 過去との比較（先月 vs 今月）

---

### 9. 🗂️ マイコレクション

**嬉しいポイント**: 自分だけの知識ベースを作れる

| 項目 | 内容 |
|------|------|
| 機能概要 | 記事の整理・ハイライト・メモ機能 |
| 実装イメージ | フォルダ/タグ管理、ハイライト保存、メモ追加 |
| 優先度 | 未定 |
| 難易度 | 中 |

**詳細**:
- 記事をフォルダ/タグで整理
- ハイライト・メモを記事に追加
- 「Claude Code設定集」「プロンプトテクニック集」など自分用まとめ

---

### 10. 📱 モバイル最適化 & オフライン保存

**嬉しいポイント**: スキマ時間に読める

| 項目 | 内容 |
|------|------|
| 機能概要 | モバイル対応とオフライン機能 |
| 実装イメージ | レスポンシブ強化、PWA、音声読み上げ |
| 優先度 | 未定 |
| 難易度 | 中〜高 |

**詳細**:
- 通勤中にスマホでサクサク読める
- 気になる記事をオフライン保存
- 音声読み上げ機能（ながら聴き）
